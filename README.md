
    # Prédicteur de Prix de Maisons

    ## Description du Projet
    Ce projet utilise l'apprentissage automatique pour prédire les prix de maisons.

    ## Étapes du Processus
    1. Chargement des Données
    2. Exploration et Visualisation
    3. Prétraitement des Données
    4. Entraînement du Modèle
    5. Évaluation et Visualisation des Résultats

    ## Bibliothèques Utilisées
    - Pandas
    - Scikit-learn
    - Matplotlib
    - Seaborn
    - Streamlit

    ## Guide d'Utilisation
    1. Installer les dépendances
    2. Lancer l'interface Streamlit
    3. Cliquer sur 'Visualiser les Données'
    4. Cliquer sur 'Entraîner le Modèle'

    ## Métriques du Modèle
    - Erreur Quadratique Moyenne (MSE)
    - Score R²
